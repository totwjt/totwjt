xcode14 在设置中未找到 component 选项没办法切换和选择 simulator

在开发uniapp应用时，遇到了无法切换和选择simulator的问题。按照uniapp官网描述App平台：模拟器安装指南在设置中未找到 component 选项没办法切换和选择simulator选项

  1. 打开 Xcode, 选择 windows -> Devices and Simulators

  2. 点击 + 按钮，选择Device Type、 OS Version

  3. 重新打开 Simulator， 可以打开模拟器了
